# 🧠 Quantum GAN for Sleep EEG Data Generation

A hybrid quantum-classical machine learning project that applies **Quantum Generative Adversarial Networks (QGAN)** to generate synthetic sleep EEG data. Built with **PennyLane** and **PyTorch**.

---

## 🚀 What This Project Does

This project takes real human sleep EEG brain signal recordings and trains a Quantum GAN to generate new synthetic sleep data that looks realistic.

**Why this matters:** Real sleep data is expensive and hard to collect. Patients need to visit hospitals, wear electrodes, and sleep overnight in a lab. Synthetic sleep data generated by a QGAN can give researchers more training data without needing more patients.

---

## ⚛️ How the QGAN Works

There are two quantum circuits competing with each other:

- **Generator** takes random noise as input and passes it through a quantum circuit using RX rotation gates and CNOT entanglement gates. It outputs fake sleep data trying to fool the Discriminator.
- **Discriminator** takes data as input and passes it through a quantum circuit using RY and RZ rotation gates. It outputs a value deciding if the data is real or fake.

They compete until the Generator creates data the Discriminator cannot distinguish from real sleep data.

---

## 📊 Dataset

- **Source:** Sleep EDF dataset, file EPCTL03.edf
- **Signal:** EEG brain recordings from a human patient during sleep
- **Processing:** Signal split into 843 epochs of 30 seconds each
- **Features per epoch:** Mean, Standard Deviation, Minimum, Maximum
- **Normalization:** All features normalized to [-1, 1] to match quantum circuit output range

---

## 🔬 Key Experimental Finding

We trained both QGAN and Classical GAN for 50 epochs and compared them.

| Metric | QGAN | Classical GAN | Winner |
|--------|------|---------------|--------|
| Mean MAE ↓ | 0.4829 | 0.0777 | Classical |
| Std MAE ↓ | 0.0693 | 0.2147 | **QGAN** |
| Generator Loss ↓ | 0.3948 | 0.6730 | **QGAN** |

**Main Finding:** Classical GAN learns the average behavior of sleep data better. QGAN learns the variability and spread of sleep data better. They have complementary strengths suggesting a hybrid approach as future work.

## 📥 Installation

1. Clone this repository:
```bash
git clone https://github.com/IkramAlgo/QGAN-Sleep-EEG.git
cd QGAN-Sleep-EEG
```

2. Create conda environment:
```bash
conda create -n qgan310 python=3.10
conda activate qgan310
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```

---

## ▶️ How to Run

**Train both QGAN and Classical GAN:**
```bash
python -m qgan.train
```

**Generate paper figures:**
```bash
python -m qgan.visualize
```
